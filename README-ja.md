# moviedownloader
これは、日本語を使用するLinuxユーザー向けにYoutubeおよびNiconicoムービーをダウンロードするShellScriptツールです。
## 動作環境
* [Ubuntu](https://www.ubuntu.com/) 16.04
## 使用しているプログラム
* [nicovideo-dl](http://sourceforge.jp/projects/nicovideo-dl/)
* [youtube-dl](http://yt-dl.org/)
